# 🍕 Pizza Sales Data Analysis

## 📌 Project Overview

This project focuses on analyzing pizza sales data to extract meaningful business insights and support data-driven decision-making. The analysis covers sales performance, target comparison, and future sales forecasting.

The goal of this project is to understand sales trends, identify key performance indicators (KPIs), and provide actionable insights to improve business strategies.

---

## 🎯 Objectives

* Analyze overall sales performance
* Compare actual sales with targets
* Identify top-selling and low-performing products
* Forecast future sales trends
* Provide insights for better business decisions

---

## 📊 Key Analysis

### 🔹 Sales Overview

* Total Revenue Analysis
* Total Orders & Quantity Sold
* Average Order Value
* Daily & Monthly Sales Trends

### 🔹 Target vs Sales Analysis

* Comparison between actual sales and targets
* Identification of performance gaps
* Highlighting high-performing periods

### 🔹 Sales Forecasting

* Time series analysis on historical data
* Trend and seasonality identification
* Future sales prediction

---

## 🛠️ Tools & Technologies

* **Excel / Power BI / Tableau** (for dashboards & visualization)
* **Python (Pandas, NumPy, Matplotlib)** *(optional)*
* Data Cleaning & Preprocessing
* Time Series Forecasting

---

## 📈 Key Insights

* Identified peak sales days and hours
* Found top-performing pizza categories and sizes
* Detected underperforming products
* Observed seasonal sales patterns
* Provided accurate future sales estimations
